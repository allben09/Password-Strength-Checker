# 🔐 Password Strength Checker

A modern, feature-rich password strength checker that provides real-time analysis, password generation, and breach detection.

![Version](https://img.shields.io/badge/Version-1.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![PRs](https://img.shields.io/badge/PRs-welcome-brightgreen)

## ✨ Features

- **Real-time Password Analysis**: Instant feedback as you type
- **Visual Strength Meter**: Color-coded progress bar (0-100%)
- **8-Point Security Checklist**:
  - Length validation (minimum 8 characters)
  - Uppercase/Lowercase letters
  - Numbers and special characters
  - Repeating pattern detection
  - Sequential pattern detection
  - Common password detection
- **🔍 Breach Detection**: Checks against Have I Been Pwned API
- **🔑 Strong Password Generator**: Creates cryptographically secure passwords
- **👁️ Show/Hide Password**: Toggle visibility for convenience
- **📋 Copy to Clipboard**: One-click password copying
- **🌓 Dark/Light Theme**: Persistent theme preference
- **📱 Fully Responsive**: Works on all devices

## 🛠️ Technologies Used

- HTML5
- CSS3 (with animations & glass-morphism)
- Vanilla JavaScript (ES6+)
- Have I Been Pwned API v3
- Web Crypto API (SHA-1 hashing)
- Font Awesome Icons
