## ✨ Features

| Feature | Description |
|---------|-------------|
| 🚀 **Real-time Analysis** | Instant feedback as you type |
| 📊 **Visual Strength Meter** | Color-coded progress bar (0-100%) |
| ✅ **8-Point Checklist** | Length, cases, numbers, special chars, patterns |
| 🔍 **Breach Detection** | Checks against Have I Been Pwned API |
| 🔑 **Password Generator** | Cryptographically secure passwords |
| 👁️ **Show/Hide** | Toggle password visibility |
| 📋 **Copy to Clipboard** | One-click copy functionality |
| 🌓 **Dark/Light Theme** | Persistent theme preference |
| 📱 **Responsive** | Works on all devices |
| ⌨️ **Keyboard Shortcuts** | `Ctrl + G` to generate |
